Setup Your Application

  1. Install Caduceus 
    pip install --upgrade caduceus
  2. Copy the following code to {{object.django_project_name}}/{{object.django_project_name}}/wsgi.py 
    """
WSGI config for '{{object.django_project_name}}' project.

It exposes the WSGI callable as a module-level variable named `application`.

For more information on this file, see
https://docs.djangoproject.com/en/1.10/howto/deployment/wsgi/
"""

import os

from caduceus.wsgi import Caduceus
from django.core.wsgi import get_wsgi_application

os.environ.setdefault("DJANGO_SETTINGS_MODULE", "{{ object.django_project_name }}.settings")

caduceus_project_id = "{{object.project_id}}"
application = Caduceus(get_wsgi_application(), caduceus_project_id)
  3. Run: 
    python manage.py runserver
If you don't want to run Caduceus, run the following: 
application = Caduceus(
    get_wsgi_application(), 
    caduceus_project_id, 
    activate=False)
